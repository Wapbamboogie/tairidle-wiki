# Contributing to the Tairidle Wiki

Anyone can contribute! This wiki lives in a public GitHub repository separate from the game code.

## How to edit a page

1. Click **"Edit this page on GitHub"** at the bottom of any page
2. If you don't have a GitHub account, [create one for free](https://github.com/signup)
3. Click the pencil ✏️ icon to edit the file
4. Make your changes in the editor
5. Scroll down and click **"Propose changes"**
6. Click **"Create pull request"**

A maintainer will review and merge your changes.

## Adding a new page

1. Fork the [tairidle-wiki](https://github.com/Wapbamboogie/tairidle-wiki) repository
2. Add a new `.md` file in the root directory
3. Add it to the sidebar in `.vitepress/config.ts`
4. Submit a pull request

## Guidelines

- Keep information accurate — verify against the game before adding
- Use the existing table format for consistency
- Don't add spoilers without a warning
